miniGRR: a template GRR

Defining new genomic resources can feel a bit abstract at first: different resource types expect different file formats (FASTA, BigWig, tabix-indexed TSV/VCF, …), coordinate conventions (0-based vs 1-based), and configuration options in genomic_resource.yaml. To make this concrete, we provide miniGRR, a small, self-contained Genomic Resource Repository on GitHub that you can use as a template.
